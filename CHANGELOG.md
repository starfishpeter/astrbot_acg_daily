# Changelog

## 0.3.0 - 2026-07-20

- 日报图片更新为编辑部画报布局：首条主推大图，后续资讯使用双列封面卡。
- AI 入选资讯会抓取原文详情页，优先使用 Open Graph 或正文首图；无图时使用分类视觉卡片。
- 支持 MyAnimeList、Anime News Network 和触乐等页面的列表图片提取；候选列表按来源轮转，减少单一站点占满日报的情况。
- 更新默认来源建议为当前网络下已验证的新闻页面和官方 RSS，移除响应不稳定的 Anime Herald 与 AnimeAnime。

## 0.2.1 - 2026-07-20

- 抓取未返回条目的资讯源时自动重试一次。
- 抓取失败日志增加响应类型和大小，便于区分空订阅与站点拦截页面。
- 增加封面下载、AstrBot 文转图和图片发送各阶段的中文运行日志。

## 0.2.0 - 2026-07-20

- 日报输出从 QQ 合并转发改为 AstrBot HTML 文转图生成的单张图文长图。
- 日报最多收录 8 条资讯；图片显示来源但不显示原文链接。
- 支持从 RSS、Atom、RDF 和普通列表页提取封面图；无法取得封面时使用分类文字卡片。
- 增加封面下载的私网地址、重定向、格式和大小限制。
- 改进二次元内容筛选，排除泛游戏、科技和成人向资讯。
- 支持 RSS 1.0/RDF 订阅源，并补充图片日报与封面提取测试。

## 0.1.1 - 2026-07-20

- 增加中文运行日志和可选的网页搜索核对功能。

## 0.1.0 - 2026-07-20

- 初始版本：从可配置资讯源抓取 ACG 新闻，经 AI 筛选和整理后发送日报。
